# EquationSolver | 方程求解器
不想自己解方程?这个程序或许能帮到你!  
这个程序可以求解一元二次方程和二元一次方程组  
## 安装
这个程序不会修改注册表和文件,请放心使用  
linux和MacOS可以使用wine或者虚拟机运行
Windows10和Windows11(64位)可以正常运行,其他系统没有测试

- 1.点击右侧 Releases 打开发行版页面  
- 2.点击最新的版本下方的 Assets  
- 3.根据自己的操作系统选择合适的文件
## 使用(大佬无视)
- 注意这个程序不能求解出未知数的精确值和虚数,如果对这方面有要求请使用专业的软件
- 这个程序全部使用命令行交互,对新手可能不太友好,以后会改进  
- 在输入以后要按下回车(Enter)键
